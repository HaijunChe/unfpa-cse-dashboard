Project 01

UNFPA SRHR Interactive Dashboard

Built as a demonstration for UNFPA CSE Programme application

Dashboard Overview with World Map and 6 Interactive Charts
Interactive Charts and Filters Panel

Interactive dashboard covering adolescent birth rates, contraceptive prevalence, HIV trends, sex education coverage, and child marriage across 75 countries. Built with a Python data pipeline and deployed as a single-page web application. Features include metric switching on choropleth map, country search with detail panel, interactive radar charts, and CSV export. Data sourced from WHO, UNFPA, UNAIDS, and World Bank.

Project 02

Air Quality & Health Risk Monitoring

Identifying non-obvious environmental signals as health risk indicators for data-scarce populations

Seasonal Pattern Analysis
Comprehensive Environment-ILI Correlation Dashboard
Predictive Lag Analysis (3-week lead time)
Model Residual Analysis

This project tracks air quality and meteorological data as early signals for influenza-like illness (ILI) outbreaks in Karachi, a city with limited health surveillance infrastructure. The key finding: temperature and NO₂ levels predict ILI cases with a 3-week lead time (r=-0.419, p=0.0001). This cross-indicator analysis — finding non-obvious linkages that standard reporting would miss — is directly transferable to CSE programme monitoring data.

Tools: Python · pandas · matplotlib · scikit-learn · Gradient Boosting · XGBoost · R

Project 03

AI Robotics for Disaster Relief

Safe motion planning for robotic arms in post-disaster rescue scenarios · Tsinghua University AI & Robot Lab

HER Framework: sparse reward learning
AC-DDPG: actor-critic obstacle planning
Results: 90%+ task success rate

Goal-conditioned reinforcement learning framework (AC-DDPG + Hindsight Experience Replay) for robotic arm planning in unpredictable disaster-relief environments. Validated across static and dynamic obstacle scenarios. Achieved 90%+ task success rate with stable convergence. The core challenge — building a system that stays robust when conditions are unpredictable — mirrors the analytical mindset needed for complex humanitarian programme data.

Tools: Python · PyTorch · MuJoCo · AC-DDPG · HER · Goal-Conditioned RL
