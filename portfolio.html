PROJECT 02

Air Quality & Health Risk Monitoring

Identifying non-obvious environmental signals as health risk indicators for data-scarce populations

Seasonal Pattern Analysis
Comprehensive Environment-ILI Correlation
Predictive Lag Analysis (3-week lead)
Model Residual Analysis

This project tracks air quality and meteorological data as early signals for influenza-like illness (ILI) outbreaks in Karachi. Key finding: temperature and NO₂ levels predict ILI cases with a 3-week lead time (r=-0.419, p=0.0001). This cross-indicator analysis is directly transferable to CSE programme monitoring data.

Python · pandas · matplotlib · scikit-learn · Gradient Boosting · XGBoost · R

Goal-conditioned reinforcement learning framework (AC-DDPG + Hindsight Experience Replay) for robotic arm planning in unpredictable disaster-relief environments. Validated across static and dynamic obstacle scenarios. Achieved 90%+ task success rate with stable convergence. The core challenge — building a system that stays robust when conditions are unpredictable — mirrors the analytical mindset needed for complex humanitarian programme data.
