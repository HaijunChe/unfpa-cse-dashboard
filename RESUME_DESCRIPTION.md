# 简历项目描述模板

## 方案一：简洁版（适合技术简历）

**UNFPA 青少年生殖健康数据仪表盘** | [Live Demo](https://haijunche.github.io/unfpa-cse-dashboard/dashboard.html) | [GitHub](https://github.com/HaijunChe/unfpa-cse-dashboard)
- 独立开发交互式数据可视化仪表盘，涵盖 75 个国家的 10 项 WHO/UNFPA 健康指标
- 使用原生 HTML/CSS/JavaScript + Plotly.js 构建，包含世界地图、趋势图、散点图等 5 种图表类型
- 实现交互功能：国家搜索、区域筛选、指标切换、点击联动详情面板、数据导出
- 零依赖部署：单 HTML 文件托管于 GitHub Pages，支持移动端响应式布局

## 方案二：详细版（适合突出数据分析能力）

**青少年性与生殖健康数据仪表盘**
- 可视化 75 个国家/10 项健康指标（生育率、避孕率、HIV 感染率等）
- 5 种图表类型：Choropleth 世界地图、趋势折线图、区域对比柱状图、相关性散点图、雷达图
- 6 项交互功能：指标切换、搜索过滤、点击联动详情面板、排序、CSV 导出、实时数据更新
- 基于 WHO Global Health Observatory、UNFPA、UNAIDS 真实数据构建
- 技术栈：Plotly.js (可视化) + Lucide Icons + 纯原生 JavaScript
- 部署：GitHub Pages，零构建工具，单文件 1250+ 行代码

## 方案三：一句话版（适合表格或紧凑布局）

**数据可视化仪表盘** — 75 国健康指标交互式分析，5 种图表类型，GitHub Pages 部署，[查看](https://haijunche.github.io/unfpa-cse-dashboard/dashboard.html)

## 截图建议

在简历中附上截图会让项目更有说服力：

1. 打开仪表盘页面
2. 按 `F12` 打开开发者工具，点击设备模拟图标，选择 "Laptop with HiDPI screen"（1440x900）
3. 截图保存整个页面或截取仪表盘上半部分（地图 + 统计卡）
4. 将图片裁剪为 16:9 比例，放在简历中
